# Parallel Rails

A music game about infinite parallel rails, written by Pixi.js.

It's just a project for practice, all musics and pictures don't belong to me.

## Build and run

```text
$ pnpm install
$ pnpm build
$ pnpm serve
```

Then open `http://localhost:3000` in your browser.
